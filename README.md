# maptalks-hash

Add URL hashes to web pages with maptalks

# Getting started

```javascript
    var mouseCoordinate = new maptalks.MouseCoordinate().addTo(map)
```

# Options

Option | Type | Default 
------ | ---- | ------- 
`position` | String or Object | { 'bottom' : 0, 'right' : 0 }
`precision` | Number | 5
